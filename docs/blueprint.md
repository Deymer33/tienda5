# **App Name**: SYS Govil Solutions

## Core Features:

- Homepage Showcase: Display key services, company info, and value proposition on the landing page.
- About Us Section: Detail the company's mission, vision, values, and expertise.
- Contact Information Display: Show the company's location, phone number, WhatsApp link/number, email, and social media links.
- Lead Generation Form: Capture user inquiries with a contact form. Use a tool to intelligently route captured emails to the best internal resource.
- Service Description Pages: Dedicated pages for each service offered (Abastecimiento, Mantenimiento, Logística, Asesoría, Proyectos).
- Social Media Integration: Enable easy access to the company's Facebook and Instagram profiles.

## Style Guidelines:

- Primary color: Deep blue (#0404E2) reflecting reliability and professionalism.
- Background color: Very light blue (#E8E8FF), subtly echoing the primary, for a clean, trustworthy feel.
- Accent color: Indigo (#2828F6), for calls to action and highlights, offering a fresh visual pop.
- Body font: 'PT Sans', a clear, readable font.
- Headline font: 'Space Grotesk', offering a clear tech-foward look that matches the font for body.
- Use professional and simple icons to represent each service.
- Clean, structured layout with clear section divisions for easy navigation.
